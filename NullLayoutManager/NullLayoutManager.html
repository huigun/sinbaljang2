Class NullLayoutManager

public class NullLayoutManager implements ComponentListener

 

이 클래스는 레이아웃 관리자로 자바 기본 라이브러리에서 지원하는 레이아웃 관리자와는 그 기능과 성질이 틀리다

전체 GUI 클래스의 레이아웃을 관리할 의도로 만들었으면 버그가 존재할 가능성이 있다 – 주의 요망

특별히 다른 레이아웃 관리자를 쓰지 않은 이상 이 클래스 하나만 있으면 전체 레이아웃을 컨트롤 할 수 있음

ComponentListener 인터페이스를 상속받아 자동으로 최상위 컨테이너의 리 사이즈를 감지하여 조정함

 

컨테이너에 더한 컴포넌트나 컨테이너의 시작 점이나 끝점은 상위 컨테이너에 의존하는 값임

즉 전체 크기가 변한 것에 상관없이 실제 상위 컨테이너에 절대값으로 달라붙음

상위 컨테이너가 움직이면 같은 간격을 유지하면서 따라서 움직임

 

내부적으로  null layout 을 통해 구현되어 있어 실제 시작점과 크기를 지정해 주어야 한다

시작점은 픽셀 값을 넣어주면 되지만 크기는 메소드에 따라서 페센트 값을 넣어 주어야 한다

 

Scrollpane 같은 클래스는 컴포넌트로 취급

 

 

               Frame f = new Frame("test");

               NullLayoutManager aa = new NullLayoutManager(f,500,500);

 

               Button b1 = new Button("button resize");

               Button b2 = new Button("button static");

              

               Panel p1 = new Panel();

                Panel p2 = new Panel();

               Panel p3 = new Panel();

 

               p1.setBackground(Color.black);

               p2.setBackground(Color.BLUE);

               p3.setBackground(Color.GRAY);

              

               aa.addContainer(p1,0,0,50,50);

               aa.addContainer(p2,250,0,50,50);

               aa.addContainer(p3,0,250,100,50);

              

               aa.addComponent(p3,b2,20,20,100,20);

               aa.addComponentResize(p3,b1,150,20,30,20);

 

               f.show();
